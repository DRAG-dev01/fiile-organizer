\# File Organizer



A desktop app that automatically sorts files in a selected folder into subfolders by file type.



\## Features

\- Select any folder using a folder picker dialog

\- Automatically sorts files into subfolders by type

\- Handles unknown file types with a Misc folder

\- Skips already organized files



\## How to run

1\. No dependencies needed

2\. Run: `python file_organizer_app.py`



\## Built with

\- Python

\- tkinter

\- os

\- shutil
